# 📊 MF Analytics Platform

A complete Mutual Fund Analytics Platform built using Python, SQL, and Power BI to analyze Indian mutual fund data. This project demonstrates an end-to-end data analytics workflow, from data extraction and cleaning to database design, exploratory data analysis, risk analysis, and interactive dashboards.

---

## 🚀 Project Overview

The objective of this project is to build a full-stack analytics platform for Indian mutual funds using publicly available data. The project follows a complete ETL (Extract, Transform, Load) pipeline and provides meaningful insights into fund performance, AUM growth, SIP trends, benchmark comparisons, and investor demographics.

---

## 📂 Dataset

The project uses publicly available mutual fund data, including:

* Fund Master
* NAV History
* AUM by Fund House
* Monthly SIP Data
* Category Inflows
* Folio Count
* Scheme Performance
* Transactions
* Portfolio Holdings
* Benchmark Data

**Data Sources**

* AMFI India
* mfapi.in
* Public NSE/BSE data (where applicable)
